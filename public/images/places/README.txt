EPIC TOUR — hero background images
=======================================

1. Put your JPG / WebP / PNG files in this folder (same folder as this file).

2. List filenames in manifest.json, for example:

   { "images": ["bali.jpg", "dubai.webp", "maldives.png"] }

3. Or use full URLs in manifest.json:

   { "images": ["https://example.com/hero1.jpg"] }

If manifest.json is empty or missing files, the site uses built-in placeholder
destination photos until you add your own.
